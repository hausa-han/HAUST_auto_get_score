此工具用来快速自动获取河南科技大学学生当前学期的有效成绩
__________________________________________________________________


使用时需导入以下模块：

import requests

import hashlib

import re

import os

其中requests是第三方库，需要用：

pip/pip3 install requests

来安装
__________________________________________________________________


使用方法：
打开run.bat并输入：

python/python3 get[TAB][CR]

初次使用会要求输入学号和教务系统密码，

之后会加密存储到文件中

从第二次使用开始均只需运行，即可得到成绩图片
